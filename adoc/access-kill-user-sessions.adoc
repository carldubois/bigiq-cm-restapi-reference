= BIG-IQ APM kill active access sessions.


[[_overview]]
== Overview
API used to kill active access sessions on BIGIP using a BIGIQ centralized management system.


=== Version information
[%hardbreaks]
_Version_ : 5.2


=== URI scheme
[%hardbreaks]
_BasePath_ : /mgmt/cm/access/tasks
_Schemes_ : HTTPS


=== Consumes

* `application/json`


=== Produces

* `application/json`
