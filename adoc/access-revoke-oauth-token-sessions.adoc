= BIG-IQ APM OAuth Token Revocation on BIG-IP devices.


[[_overview]]
== Overview
API for OAuth Token Revocation on BIG-IP devices using a BIG-IQ centralized management system.


=== Version information
[%hardbreaks]
_Version_ : 5.2


=== URI scheme
[%hardbreaks]
_BasePath_ : /mgmt/cm/access/tasks
_Schemes_ : HTTPS


=== Consumes

* `application/json`


=== Produces

* `application/json`
